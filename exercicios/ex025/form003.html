Anotações

Input com type text: recebe textos e números.

Input com type password: recebe senhas, ocultando os caracteres.

Input com type month: recebe mês e ano, disponibiliza calendário do navegador e não aceita datas inválidas.

Input com type date: recebe data completa 05/09/2022, disponibiliza calendário do navegador e não aceita datas inválidas.

Input com type time: recebe horas, disponibiliza relógio do navegador, não aceita horas inválidas.

Input com type number: recebe números, possui atributos:

  • min: valor mínimo
  • max: valor máximo
  • step: passo
