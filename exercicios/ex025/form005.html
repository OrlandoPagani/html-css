Anotações

Inputs do tipo radio:

  • Agrupar dentro de um fieldset para melhor organização
  • Para que seja possível selecionar apenas uma opção, utilizar o mesmo valor no atributo 'name'
  • Utilizar valores diferentes no atributo id
  • Adicionar o atributo value em cada opção, dessa forma, o valor dentro do atributo 'value' será transminido para o form
  • Envelopar cada opção com uma label, carregando o mesmo id no atributo 'for' para que quando clicado em um texto, seja selecionada a opção também

Inputs do tipo checkbox:

  • Utilizar o atributo type='checkbox'
  • Utilizar nomes e ids diferentes para cada opção
  • Envelopar cada opção com uma label, carregando o mesmo id no atributo 'for' para que quando clicado em um texto, seja selecionada a opção também
  • Utilizar o tributo 'checked' para definir uma opção como selecionada
